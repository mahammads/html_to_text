HTML text scrapper  
basic setup:
- install python= 3.8 on your machine

- create virtual environment for this project

- activate virtual environment

- install all dependancies using command = pip install -r requirements.txt

- set your project folder path in config.py

- if code running for single html file then set flag = True in config.py file.

- if multiple file present in folder then user can add start and end keyword in config.py file as mentioned format and set flag = False.

- put your html file into html_file folder.

- to extract the text from html file run the html_scrapper.py file.

- code will ask for user input to enter start and end keyword for filter the text.

- output will save in output.txt in root folder.
